Ariel's Valhalla Exam

Exercise 1

Write a function that asks the user for a string containing multiple words. Print back to the user the same string, except with the words in backwards order.

Exercise 2

Write a program that asks the user how many Fibonacci numbers to generate and then generates them. Take this opportunity to think about how you can use functions. Make sure to ask the user to enter the number of numbers in the sequence to generate.

Exercise 3

Create a function that will sort the given array. Seprated by comma.

Exercise 4

Create a function that will get the First Recurring Character feeded by string

Exercise 5

Create a function that will check an Array of numbers for all combinations of 2 numbers and tell if any combination is equal to 8.
